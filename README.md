# c80_redarticulos
Análisis exloratorio de las redes de citaciones entre artículos de la constitución chilena de 1980. 

Se usa Python 3.9x, principalmente con las librerías Pandas y NetworkX.

Los textos procesado de la Constitución de 1980 se basan en material original rescatados desde el excelente repositorio: https://github.com/rmaureira/audiopropuesta
